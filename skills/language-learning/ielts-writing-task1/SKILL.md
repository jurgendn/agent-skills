# IELTS Writing Task 1 Coach

Task 1 is a 20-minute, minimum-150-word academic report. It is not an essay — no opinion, no personal examples. The task is to select, describe, and compare the most significant features of the given visual(s).

## Use this when

- The user shares a Task 1 prompt (graph, chart, table, diagram, process, map) and wants a draft or critique.
- The user wants a band estimate with actionable feedback.
- The user asks how to write an overview, select key features, or describe trends.
- The user wants vocabulary or grammar for data description.

## Do not use this when

- The user is working on Task 2 (argumentative essays). Use `ielts-writing-task2`.
- The user is asking about Speaking. Use `ielts-speaking-coach`.
- The user is studying for General Training Task 1 (letter writing) — note the difference if they mention it.

## Band Descriptors (Task 1)

Scores are averaged equally across four criteria. Task 2 is weighted more heavily in the final Writing band, but Task 1 is still significant.

| 8 | Covers requirements; presents and highlights key features clearly with appropriate selection; data accurately described; summary adequate, accurate, and relevant |

| 5 | Generally addresses requirements but format may be inappropriate; unclear overview; may include irrelevant, inappropriate, or inaccurate content |
| 4 | Partially addresses requirements; key features inadequately covered; may be inaccurate or format inappropriate |
| 3 | Fails to address requirements; no overview; limited or irrelevant content |

## Workflow

### Step 1 — Identify the visual type and data

Determine:
- Type: line graph / bar chart / pie chart / table / map (before & after) / process diagram / multiple charts
- Time dimension: static snapshot vs. change over time
- Key entities: what is being compared (countries, years, categories, stages)
- Overall trend or pattern: what is the single most important thing the data shows

### Step 2 — Plan the response structure

Standard Task 1 structure (4 paragraphs):

1. **Introduction** — Paraphrase the prompt (do NOT copy it). One or two sentences describing what the visual shows.
2. **Overview** — The 2–3 most significant features. No data. This is the most critical paragraph for Task Achievement.
3. **Body paragraph 1** — Specific details with data for the first main group/trend.
4. **Body paragraph 2** — Specific details with data for the second main group/trend.

For processes or maps, adapt to: Introduction → Overview of stages/changes → Detailed description of each phase.

### Step 3 — Draft or critique

When drafting:
- Write the overview before any data — it forces selection of key features.
- Use comparative and trend language precisely.
- Avoid mechanical listing of every data point.
- Stay at ≥150 words (examiners penalise under-length).

When critiquing a user's response:
- Check for presence and quality of overview first.
- Check that key features (not all features) are selected.
- Check data accuracy against the prompt.
- Then evaluate CC, LR, GRA.

### Step 4 — Score and feedback

Report a band estimate for each criterion and an overall estimated band:

```
Task Achievement:     X / 9
Coherence & Cohesion: X / 9
Lexical Resource:     X / 9
Grammatical Range:    X / 9
Estimated overall:    X.X / 9
```

For each criterion below 7, give 2–3 specific, actionable improvements with examples drawn from the user's actual text.

## Key language banks

### Describing trends (line graphs, time series)

- **Increase**: rose, climbed, grew, surged, jumped, increased, went up
- **Decrease**: fell, dropped, declined, decreased, plummeted, dipped
- **Stability**: remained stable, levelled off, plateaued, stayed constant
- **Fluctuation**: fluctuated, varied, oscillated
- **Peak/trough**: reached a peak of, hit a high of, bottomed out at, reached a low of

Adverbs: sharply, steeply, dramatically, significantly, gradually, steadily, marginally, slightly

### Comparing (bar charts, tables, pie charts)

- X was significantly higher than Y
- The proportion of X was roughly double that of Y
- X accounted for the largest share at Z%
- In contrast / by comparison / whereas / while

### Describing processes

- First / Initially / To begin with
- Subsequently / Following this / The next stage involves
- Finally / Ultimately / The process concludes with
- is converted into / is fed into / passes through / undergoes

### Describing maps

- To the north/south/east/west of
- Adjacent to / opposite / alongside
- was replaced by / was demolished / a new X was constructed
- The area to the left underwent significant development

## Common mistakes to flag

1. **No overview** — biggest single TA error; must be there and must be general (no numbers).
2. **Copied introduction** — paraphrase the prompt, do not reproduce it.
3. **Describing every data point** — select the most significant; don't just list.
4. **Inaccurate data** — misreading values; always recheck numbers against the prompt.
5. **Personal opinion** — "I think this shows…" has no place in Task 1.
6. **Under 150 words** — automatic penalty to TA.

## Output format

When producing a model answer:
- Label each paragraph (Introduction / Overview / Body 1 / Body 2) so the user can see the structure.
- Provide a word count.
- After the answer, give a brief note on the key structural and language choices made.

When giving feedback on the user's response:
- Quote the user's original text when identifying issues.
- Provide a corrected or improved version of the problem sentence.
- End with the scored table and top priorities.
