# External programs — structural logic & verify-live checklists

For Dimension 2 (target/sponsor fit) and Dimension 9 (mission/return-home) when the
target is a scholarship or fellowship. This file carries **only the stable
structural logic** of each program — what it weights, its mission/return-home/
mobility orientation, and the *kinds* of documents it requires. It deliberately
carries **no numbers** (deadlines, score minimums, CV/SOP length limits, stipend
amounts) because those change every cycle.

> **Hard rule.** Before using any cutoff, length limit, deadline, eligible-country
> list, or funding figure, **verify it against the live official call** at the
> program's site. A stale cached limit is worse than none. Each program below ends
> with a verify-live checklist; fill it from the current call, not from memory.

## How to use

1. Identify the named sponsor; if unnamed, ask before scoring Dimensions 2/9.
2. Read the program's stable orientation here to set what Dimensions 2 and 9
   reward.
3. Run the program's verify-live checklist against the official site and record
   each confirmed value in the scorecard's **Live-verification checklist**.

## VEF 2.0 (primary)

Site: <https://vef2.org/>

**Orientation (stable):** supports Vietnamese applicants pursuing advanced study;
weights **academic excellence + leadership + return-home / development impact**
together — excellence alone is not the whole bar. Mission fit and a credible
contribution-to-Vietnam narrative are scored heavily (Dimension 9).

**Document types (stable kinds):** application essays/statements, CV, academic
records, references; **the CV and SOP/essays have strict length limits** that are
set per cycle — **fetch the current limits live** and treat exceeding them as a
Major/Critical readiness flag.

**Scoring implication:** for VEF, weight Dimension 2 (sponsor fit) and Dimension 9
(mission/return-home) highest; a strong but mission-generic dossier is at most
"Strong," not "Top." A reused generic SOP that ignores the mission is **Critical**.

**Verify-live checklist:**
- [ ] Current **CV length limit** and **SOP/essay length limit(s)**
- [ ] Current **deadline(s)** and submission portal
- [ ] **Eligibility** (degree stage, field scope, nationality/residency)
- [ ] Required **document set** and any prescribed essay prompts
- [ ] Any **return-home / service** commitment terms
- [ ] Any **test-score** expectations

## DAAD

Site: <https://daad.de>

**Orientation (stable):** German academic exchange funder; **program-specific** —
DAAD runs many distinct scholarships, each with its own rules. Generally values
academic merit, a clear and feasible study/research plan, and motivation for
studying in Germany; some lines have development/return or home-country dimensions.

**Document types (stable kinds):** CV, motivation/research proposal, transcripts,
references, often proof of host/supervisor contact for research grants.

**Scoring implication:** do not assume one DAAD rubric — identify the **specific
DAAD program** first, then set Dimension 2 to that line's stated priorities.

**Verify-live checklist:**
- [ ] **Which DAAD program** and its specific objectives
- [ ] Required documents, **length limits**, and proposal format
- [ ] Whether a **host/supervisor confirmation** is required
- [ ] Deadlines, eligibility, language/test requirements

## Fulbright

Sites: <https://fulbright.gov> · Vietnam: <https://vn.usembassy.gov>

**Orientation (stable):** US government exchange program with a **mutual-
understanding / exchange mission** and, for many tracks, a **home-country return /
two-year home-residency** orientation; administered per country, so Vietnam-specific
rules apply for Vietnamese applicants.

**Document types (stable kinds):** study/research objective, personal statement, CV,
transcripts, references; country-specific eligibility and process.

**Scoring implication:** weight the exchange-mission and return orientation in
Dimensions 2/9; verify the **Vietnam-specific** call, which can differ from the
global program.

**Verify-live checklist:**
- [ ] **Vietnam-specific** eligibility, fields, and process (US Embassy Vietnam)
- [ ] Required statements and **length limits**
- [ ] **Return / home-residency** terms
- [ ] Deadlines, test-score and reference requirements

## Erasmus Mundus / EMJM

Site: <https://erasmus-plus.ec.europa.eu> (EACEA catalogue of programmes)

**Orientation (stable):** **Erasmus Mundus Joint Master's (EMJM)** are
**consortium, multi-country** programs; **mobility** (studying in ≥2 countries) is
structural, and selection is run by the **individual consortium**, each with its own
academic criteria. Fit is to a *specific joint program*, not a single university.

**Document types (stable kinds):** per-consortium application — CV, motivation
letter, transcripts, references, language proof; each catalogue entry differs.

**Scoring implication:** Dimension 2 fit is to the **named EMJM consortium's**
stated profile and the mobility/cohort logic; there is no single Erasmus rubric.

**Verify-live checklist:**
- [ ] The **specific EMJM programme** in the EACEA catalogue and its criteria
- [ ] **Mobility** structure and which countries/partners
- [ ] Required documents, **length limits**, language requirements
- [ ] Deadlines and the consortium's selection process

## Marie Skłodowska-Curie Actions (MSCA)

Site: <https://marie-sklodowska-curie-actions.ec.europa.eu>

**Orientation (stable):** EU research-funding actions emphasizing **research
excellence, training, and mobility**; **doctoral networks** and **postdoctoral
fellowships** are distinct instruments with different applicant roles. Often the
*host/supervisor and proposal* are central, and a **mobility rule** (not having
resided in the host country beyond a set recent window) is structural.

**Document types (stable kinds):** research proposal, CV, host/supervisor and
institution details; for fellowships the proposal is typically co-developed with a
host.

**Scoring implication:** Dimension 2/8 fit centers on the **host and proposal**; the
mobility rule can be a Dimension 1 gate — **verify the current mobility window and
the specific MSCA instrument**.

**Verify-live checklist:**
- [ ] **Which MSCA instrument** (Doctoral Networks vs. Postdoctoral Fellowships, etc.)
- [ ] Current **mobility rule** window and reference period
- [ ] **Host/supervisor** requirement and proposal format/limits
- [ ] Deadlines, eligibility, and evaluation criteria weights
