# Failure patterns — red-flag library

Feeds Dimension 11 (risk & readiness) and informs Dimension 2/9/10 scoring. Each
flag is tagged by severity with a concrete fix. Severity drives the readiness band
in `rubric-descriptors.md`:

- **Critical** — blocks submission / caps the dossier (forces D11 ≈ 1; often D2 or
  D1 too). Must fix before submitting.
- **Major** — materially lowers the dossier; fix before submission (forces D11 ≈ 3).
- **Minor** — cosmetic or marginal; fix if time allows.

> No cycle-specific numbers here. "Exceeds the limit" means exceeds the **live
> official limit** (see `external-programs.md`), not a cached number.

## Cross-cutting (all target types)

- **Ineligible as-is, unaddressed** — degree/stage/mobility rule unmet with no
  remediation path. *(Critical)* → resolve eligibility (see
  `degree-equivalence.md`) or change target.
- **Documents describe different applicants** — SOP, CV, and letters imply
  different theses or strengths. *(Critical/Major by degree)* → unify around one
  research story (Dimension 10; `assets/coherence-checklist.md`).
- **Missing required document type** — a document the call requires is absent.
  *(Critical)* → obtain it; verify the required set against the live call.
- **Exceeds the live length/format limit** — CV/SOP/essay over the official limit.
  *(Major; Critical for limit-strict sponsors like VEF)* → cut to the verified
  limit.
- **SOP claim not backed by CV** — a stated achievement has no CV evidence.
  *(Major)* → add evidence or drop the claim.
- **CV strength unused in narrative** — the strongest evidence never appears in the
  SOP. *(Minor/Major)* → surface it in the narrative.
- **Unexplained transcript anomaly** — a dip/gap/low prerequisite with no context.
  *(Major)* → address briefly and factually.
- **Hallucinated or unverifiable claim** — a paper/venue/role that cannot be
  verified. *(Critical)* → remove or correct; never overstate authorship.
- **Stale or generic faculty/program fit** — fit to old work or by keyword only.
  *(Major)* → re-ground in recent work, fetched live.

## Type A — Research PhD

- **Manager-only / character-only letters for a research PhD** — no one attests
  research ability. *(Major)* → secure at least one research-supervisor letter.
- **No first-author / lead contribution claimed where the target expects research
  readiness** — only middle-author or coursework. *(Major)* → foreground genuine
  ownership; if none, reframe target (e.g. master's-first) or strengthen first.
- **Publication strength asserted by count, not venue** — "5 papers" all
  workshop/non-archival. *(Minor/Major)* → present tier + role honestly
  (`venue-quality.md`); do not pad.
- **No specific faculty fit** — SOP names a department but no people/recent work.
  *(Major)* → add concrete, current fit.

## Type B — Research / coursework Master's

- **Weak core prerequisites unaddressed** — soft grades in courses the program
  builds on. *(Major)* → contextualize and/or show later mastery.
- **No research thesis in the SOP** — personal-history essay with no direction.
  *(Major)* → state a credible intended direction the master's serves.
- **Over-claims research output expected of a PhD applicant** — inflates a course
  project into a "publication." *(Major)* → describe it accurately as preparation.

## Type C — Scholarship / Fellowship

- **Scholarship essay reuses a generic SOP and ignores the sponsor's mission** —
  no return-home/development/mission engagement. *(Critical)* → rewrite to the
  sponsor's stated mission and prompts.
- **Mission/return-home stated as intent with no prior action** — claims impact
  but the record shows none. *(Major)* → evidence it with concrete prior
  leadership/service, or temper the claim.
- **Leadership section is participation-only** — lists membership, not outcomes.
  *(Minor/Major)* → show roles and results.
- **Ignores sponsor-specific eligibility/format** — wrong document set or over the
  verified length limit. *(Critical)* → conform to the live call
  (`external-programs.md`).

## Type D — Research internship / pre-doc / RA

- **No artifact demonstrating hands-on depth** — claims skills with nothing to show.
  *(Major)* → add a concrete artifact in the lab's area (code, implementation,
  result).
- **Fit stated at keyword level to the lab** — no engagement with the group's
  recent work. *(Major)* → ground fit in specific recent outputs, fetched live.
- **No letter that can vouch for hands-on competence** — only distant/course
  letters. *(Major)* → add a recommender who supervised relevant hands-on work.
