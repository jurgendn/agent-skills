# Exercise Templates by Domain

Fill these in with the learner's actual source material. Each template lists shapes per tier so a problem hits the intended cognitive level. Keep one concept per problem and make every problem answerable from what the learner studied.

## Conceptual / qualitative topics

- **T1:** "Define ___." / "Which of these is an instance of ___?" / "State the assumption ___ relies on."
- **T2:** "Walk through what ___ does on this concrete scenario." / "Predict the qualitative outcome when ___ changes."
- **T3:** "Why does ___ exist instead of just using ___?" / "Under what condition does ___ stop working, and why?" / "Compare ___ and ___: what does each buy you?"
- **T4:** "Describe a real situation where ___ would mislead you, and what you'd use instead." / "Critique this claim about ___." / "Extend ___ to handle case ___."

## Mathematical / derivation topics

- **T1:** "State the definition / theorem / identity." / "Name the conditions under which it holds."
- **T2:** "Evaluate ___ for these specific values." / "Apply the formula to this case and simplify."
- **T3:** "Which hypothesis of the theorem fails for this input, and what breaks?" / "Why is the bound this and not tighter?" / "Explain the role of term ___ in the expression."
- **T4:** "Derive ___ from first principles." / "Construct a counterexample showing the assumption is necessary." / "Generalize the result to ___ and state where the proof changes."

## Coding / implementation topics

- **T1:** "What does function ___ return for this input?" / "Identify which line implements ___."
- **T2:** "Trace this code on this input; give intermediate state." / "Fill in the missing line so the output is ___."
- **T3:** "This implementation is correct but slow / subtly wrong — diagnose why." / "Why is the API designed this way rather than ___?"
- **T4:** "Implement ___ from the spec." / "Find an input that triggers the bug / edge case." / "Refactor ___ to support ___ and justify the design."

## Analysis / paper-critique topics

- **T1:** "What problem does this paper claim to solve?" / "Name the baseline it compares against."
- **T2:** "Read Table N: which method wins on metric M, and by how much?" / "Restate the main result in one sentence."
- **T3:** "Which experiment would you need to believe the central claim?" / "What confound does the ablation rule out — and what does it not?"
- **T4:** "Design the experiment that would falsify the paper's claim." / "Propose the strongest follow-up the paper enables." / "Where is the gap a reviewer would attack?"

## Quality bar for every problem

- Unambiguous: exactly one defensible answer (or a clearly bounded set).
- Self-contained: all needed data is in the problem or the named source.
- Tier-honest: a problem tagged Tier 3 must require reasoning, not recall in disguise.
- Faithful: never require a fact the learner's source material didn't contain.
