# Search Log

Record how the reading set was built. This prevents the map from being quietly
shaped by one keyword, one recommender system, or one LLM-generated list.

## Query / Trail Template

```markdown
## YYYY-MM-DD — <topic or subtopic>

- Goal:
- Search mode: keyword | semantic | citation trail | survey trail | recommender | LLM-assisted | advisor/user seed
- Query / seed source:
- Inclusion rule:
- Exclusion rule:
- Sources added:
- Sources rejected:
- What this search likely missed:
- Next search:
```

## Coverage Checks

- Have you searched both method names and problem names?
- Have you translated vocabulary across neighboring disciplines?
- Have you followed at least one citation trail backward to a canonical source?
- Have you followed at least one recent citation trail forward?
- Have you added any dissenting, negative, or limitation-focused sources?
- If an AI/recommender suggested papers, have you searched outside that cluster?
